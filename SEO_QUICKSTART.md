## üÜò Troubleshooting

### "My pages aren't indexed"
- **Wait**: Indexing takes 1-7 days
- **Check**: Google Search Console > Coverage
- **Fix**: Submit sitemap again in GSC

### "Not ranking for my keywords"
- **Normal**: Takes 3-6 months to rank
- **Do**: Build backlinks (LinkedIn, GitHub, job boards)
- **Check**: Search Console > Performance to see impressions

### "Low organic traffic"
- **Options**:
  1. Add more content (blog posts, case studies)
  2. Build more backlinks (LinkedIn posts, guest posts)
  3. Update meta descriptions to improve CTR
  4. Add internal links between pages

## üéì SEO Concepts Explained Simply

### Keyword
A search term people use (e.g., "IT Project Manager Romania")

### Meta Description
The 155-character summary people see below your title in Google

### Backlink
A link from another website to yours (helps with rankings)

### Core Web Vitals
Speed metrics Google cares about (LCP, FID, CLS)

### Structured Data
Information in code that helps Google understand your page (Schema.org)

### Sitemap
A map of all pages on your site for search engines

### 301 Redirect
Permanent URL change (helps preserve rankings when moving)
