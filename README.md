# Marian Dumitru — CV Online & Portofoliu Profesional

Proiect web pentru prezentare profesionala, construit ca **CV interactiv + portofoliu tehnic**, orientat pe angajare in roluri de:
- IT Project / Digital Operations
- Technical Consultant
- Product Delivery / Integration Support
- UX/UI workflow (Figma) + implementare frontend/backend

## Obiectiv
Acest proiect functioneaza ca o **carte de vizita profesionala online**: experienta, competente, studii, certificari si proiecte demonstrabile, intr-un format modern, usor de parcurs de HR si manageri tehnici.

## Structura proiectului
- `index.html` — CV premium, interactiv, optimizat pentru prezentare profesionala.
- `projects/index.html` — hub de portofoliu, cu proiecte filtrabile pe arii tehnice.
- `projects/*.html` — pagini de proiect (case studies + demo vizual).
- `assets/styles.css` — identitate vizuala premium, responsive, animatii.
- `assets/app.js` — interactiuni UI (reveal, filtre) + export PDF CV.

## Functionalitati principale
- CV online modern, cu sectiuni clare pentru HR:
  - profil profesional
  - competente cheie
  - experienta
  - studii & certificari
  - proiecte relevante
- Portofoliu cu proiecte pe tehnologii moderne:
  - JavaScript, Python, Java, Node.js, CSS
  - Mobile workflow
  - AI automation, REST APIs, webhooks, data pipelines
- Buton de apel rapid (telefon) fix pe ecran.
- Export CV in PDF, intr-o singura pagina A4, cu data si ora generarii.

## Publicare (GitHub Pages)
1. Creeaza un repository nou.
2. Urca continutul acestui folder in root.
3. In GitHub: `Settings` -> `Pages` -> `Deploy from branch` -> `main` / `/ (root)`.
4. Dupa publicare, foloseste URL-ul ca link oficial CV.

## Recomandari de utilizare pentru angajare
- Include URL-ul acestui site in:
  - LinkedIn
  - profiluri de job boards (eJobs, BestJobs, etc.)
  - mesajele de aplicare
- Pastreaza proiectele si metricile actualizate periodic.
- Foloseste formulare scurte, orientate pe impact (problem -> actiune -> rezultat).
